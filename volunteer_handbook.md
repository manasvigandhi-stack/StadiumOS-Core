# Volunteer Handbook

Welcome to the volunteer team! This handbook outlines the core protocols, expectations, and guidelines for all volunteers.

## Core Protocols

### Lost Child Protocol
In the event of a lost child, please follow these steps:
1. **Remain calm** and reassure the parents/guardians or the child if you have found them.
2. **Contact security/event coordinators** immediately with a detailed description of the child (age, clothing, hair color, name if known) and your location.
3. **Stay in place** if you have found the child, unless instructed otherwise by security.
4. **Do not broadcast** the child's full name over public PA systems if possible; use descriptions instead to maintain privacy and safety.
